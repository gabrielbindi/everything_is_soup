# Welcome to "Everything is soup"
A Linux-Config that changes the standard Linux commands into soup related vocabulary. 

## The Idea:

Everything in existence is part of one big soup.... the universe.
That's why we are building this config for our users, to build awareness, that we are part of one big **soup**. 

## What else?:

We have plug-ins installed, that automatically come with our config.

Right now we offer:

- lazyGit
- lazyDocker

## How to install:

Type in this command:
'''bash
curl -sSL "https://raw.githubusercontent.com/gabrielbindi/everything_is_soup/shell/install.sh" | bash
'''
